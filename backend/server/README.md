# ShopHub Backend

## Start the backend
```bash
node server/index.js
```

## API base URL
```text
http://localhost:5000/api
```

## Notes
- The server expects a MySQL database named `shophub_ecommerce`.
- Import the SQL schema from [database/shophub_ecommerce.sql](../database/shophub_ecommerce.sql).
- Update [.env](../.env) with your MySQL credentials.
